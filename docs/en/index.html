REVIEWED ANSWER

One answer, four auditable layers

This compact example comes from reviewed case AQ001. The full evaluation stores the question, answer, video mapping, required points, and forbidden claims.

Question

I am late in the rear court and cannot complete my normal preparation. How can I still clear to the baseline?

01

Set the boundary

Under genuine pressure, do not force an active preparation or full turn. Forehand overhead and backhand remain separate branches.

02

Give movement cues

Move the racket behind the body earlier from below. Shorten the motion as contact becomes lower and farther behind.

03

Turn it into practice

Start with shadow transitions, then deepen feeds while checking recognition, frame position, and retained racket speed.

04

Locate the evidence

00:27–00:40 for the preparation path; 01:54–02:03 for earlier contact.

FROM VIDEO TO ANSWER

Build the evidence before generating advice

  1. 01Discover and classify

    Separate teaching content from ads and non-teaching posts.

  2. 02Transcribe and review

    Use locatable transcripts or reviewed visual summaries.

  3. 03Index boundaries

    Model technique, court area, role, and active/passive state.

  4. 04Recall and filter

    Find broadly, then remove neighboring techniques and conflicts.

  5. 05Answer and learn

    Cite auditable evidence; convert confirmed errors into regressions.

QUICK START

Install the stable release

The installer runs its own doctor check, then atomically replaces an older version. Restart Codex when it finishes.

  • Python 3.10 or newer
  • No OpenAI API key
  • No transcription dependencies
Terminal
curl -L https://github.com/MuyuanGuo/badminton-skills-coach/releases/download/v1.3.0/liuhui-badminton-coach-1.3.0.zip \
  -o /tmp/liuhui-badminton-coach-1.3.0.zip
curl -L https://github.com/MuyuanGuo/badminton-skills-coach/releases/download/v1.3.0/SHA256SUMS.txt \
  -o /tmp/SHA256SUMS.txt
(cd /tmp && shasum -a 256 -c SHA256SUMS.txt)
install_dir="$(mktemp -d)"
unzip -q /tmp/liuhui-badminton-coach-1.3.0.zip -d "$install_dir"
python3 "$install_dir/liuhui-badminton-coach/scripts/install.py"
Then run doctor and restart Codex

QUALITY CONTRACT

Metrics need boundaries. So do answers.

The current baseline describes a reviewed corpus and regression set, not every possible natural-language question.

A

Source fidelity

Specific coaching claims require teaching notes or timestamped evidence.

B

Scenario separation

Handedness, court area, format, pressure state, and roles stay distinct.

C

Reviewed feedback

User feedback is verified before it affects ranking or regression rules.

D

Verifiable releases

Formal packages provide SHA-256, a CycloneDX SBOM, and build provenance.
