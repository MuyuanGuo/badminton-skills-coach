用户问题

后场被动来不及架拍，怎么把球打到底线？

01

先判断边界

真正被动时，不强求主动球的大架拍和完整侧身；正手头顶与反手需要分开处理。

02

再给动作线索

拍子更早从身体下方让到身后。击球点越低、越靠后，准备和挥拍幅度也要缩短。

03

转成训练任务

从无球转换开始，再逐渐增加喂球深度；每球只检查判断、框架和缩短后的拍速。

04

指出证据位置

00:27–00:40 看从下方到身后的路线；01:54–02:03 看提前击球。

FROM VIDEO TO ANSWER

先建立证据，再生成答案

  1. 01发现与分类

    增量观察公开视频，区分教学、广告与非教学内容。

  2. 02转写与复核

    优先使用可定位转写；语音不可靠时使用人工视觉摘要。

  3. 03索引与边界

    建立主题、场区、角色、主动被动和动作关系。

  4. 04召回与冲突过滤

    广泛找候选，再排除相邻动作、错误场景和硬负例。

  5. 05回答与反馈

    输出可审计引用；确认后的错误进入回归评测。

安装稳定版

安装器会先运行 doctor，再原子替换旧版本。完成后重新启动 Codex。

  • Python 3.10 或更新版本
  • 无需 OpenAI API Key
  • 无需安装转写依赖
Terminal
curl -L https://github.com/MuyuanGuo/badminton-skills-coach/releases/download/v1.3.0/liuhui-badminton-coach-v1.3.0.zip \
  -o /tmp/liuhui-badminton-coach-v1.3.0.zip
curl -L https://github.com/MuyuanGuo/badminton-skills-coach/releases/download/v1.3.0/SHA256SUMS.txt \
  -o /tmp/SHA256SUMS.txt
(cd /tmp && shasum -a 256 -c SHA256SUMS.txt)
install_dir="$(mktemp -d)"
unzip -q /tmp/liuhui-badminton-coach-v1.3.0.zip -d "$install_dir"
python3 "$install_dir/liuhui-badminton-coach/scripts/install.py"
随后运行 doctor 并重新启动 Codex
